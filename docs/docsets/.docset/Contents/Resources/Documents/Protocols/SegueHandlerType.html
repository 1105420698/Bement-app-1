SegueHandlerType

protocol SegueHandlerType

Use in view controllers:

1) Have view controller conform to SegueHandlerType 2) Add enum SegueIdentifier: String { } to conformance 3) Manual segues are trigged by performSegue(with:sender:) 4) prepare(for:sender:) does a switch segueIdentifier(for: segue) to select the appropriate segue case
